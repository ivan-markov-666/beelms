# Стратегия за издания и версиониране

## Семантично версиониране
Следваме [SemVer](https://semver.org/) за всички публични API.

## Модел на клоновете
- `main`: готов за продукция
- `release/*`: стабилизационни клонове
- `feat/*` / `fix/*`: краткоживеещи клонове за функционалности или поправки

## График на изданията
Седмично издание всеки **вторник 14:00 UTC**, освен при критичен хотфикс.

## Качествени изисквания (Quality Gates)
- ✅ Lint и unit тестовете преминават успешно
- ✅ E2E тестове
- ✅ Lighthouse резултат ≥ 90

## Тагване
Анотиран git таг `v<major>.<minor>.<patch>`.
