# 1.1: Инициализация на NestJS проект

## Описание
Инициализиране на нов NestJS проект с всички необходими конфигурации и зависимости.

## Технически детайли
- Инициализиране на нов NestJS проект с `nest new`
- Настройка на TypeScript в strict режим
- Конфигуриране на ESLint и Prettier за код стил
- Настройка на Jest за тестове
- Интеграция на JWT за автентикация
- Конфигуриране на HTTPS с TLS 1.3
- Интеграция със Sentry за мониторинг
- Тестове, които да тестват имплементациите (където е възможно). Тестовете трябва да се включат в regression suite.

## Изходни данни
- Инициализиран NestJS проект
- Конфигурационни файлове за TypeScript, ESLint, Prettier, Jest
- Основна структура на приложението
- Конфигурация за разработка и продукция

## Критерии за приемане
- [ ] `npm run start:dev` стартира приложението успешно
- [ ] HTTPS е конфигуриран с TLS 1.3
- [ ] Времето за стартиране е под 5 секунди
- [ ] Sentry е интегриран и работи
- [ ] Всички тестове минават успешно

## Зависимости
- Няма

## Свързани файлове
- `package.json`
- `tsconfig.json`
- `.eslintrc.js`
- `.prettierrc`
- `jest.config.js`
