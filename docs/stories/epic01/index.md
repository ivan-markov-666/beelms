## Ключови изисквания

### Сигурност
- Криптиране на чувствителни данни с AES-256
- Хеширане на пароли с Argon2id
- JWT токени с RSA ключове и механизъм за ротация
- Rate limiting за защита срещу атаки
- Защита срещу CSRF и XSS

### Наблюдаемост
- Интеграция със Sentry за мониторинг на грешки
- Метрики за време на отговор и грешки
- Детайлни логове с X-Request-ID

### GDPR съвместимост
- Криптиране на лични данни
- Възможност за експорт и изтриване на данни
- Логване на достъп до лични данни

## Технически стек
- Backend: NestJS, TypeScript
- База данни: PostgreSQL
- Кеш: Redis
- Мониторинг: Sentry, Prometheus, Grafana
- Документация: Swagger/OpenAPI
