# Learning Management System (LMS) - Документация

## За проекта
Това е начална документация за Learning Management System (LMS) - система за управление на обучение. Този документ ще бъде актуализиран с напредъка на разработката.

## Първи стъпки

### Изисквания
- Node.js (версия 16.x или по-нова)
- npm (версия 8.x или по-нова)
- Git

### Клониране на хранилището
```bash
git clone [repository-url]
cd lms
```

## Технически стек (планиран)

### Backend (предложен)
- Node.js с TypeScript
- Express или NestJS
- PostgreSQL за база данни
- JWT за автентикация

### Frontend (предложен)
- React с TypeScript
- Material-UI или подобна UI библиотека
- React Query или подобно решение за управление на състоянието

## Настройка на средата

### Конфигурационен файл
Създайте `.env` файл в основната директория с необходимите променливи за средата. Примерна структура:

```env
# Основна конфигурация
NODE_ENV=development
PORT=3000

# Конфигурация на базата данни (ще бъде добавена по-късно)
# DB_HOST=localhost
# DB_PORT=5432
# DB_USERNAME=
# DB_PASSWORD=
# DB_NAME=
```

## Архитектурни решения

### Монолитен дизайн (начален етап)
В началния етап системата ще бъде разработена като монолитно приложение с ясно разделение на слоеве:

1. **Презентационен слой** - Отговаря за потребителския интерфейс
2. **Бизнес логика** - Изпълнява бизнес правилата
3. **Достъп до данни** - Комуникира с базата данни

### Бъдеща архитектура (микросервиси)
В бъдеще системата може да бъде разделена на следните микросервиси:
1. Потребителски сървис
2. Сервиз за курсове
3. Сервиз за задачи и предавания
4. Сервиз за оценяване

Приемането на приноси към този проект ще бъде възможно след като бъде установена основната структура. Очаквайте повече информация в бъдещи актуализации.
