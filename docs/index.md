# Learning Management System (LMS) - Документация

## За проекта
Това е начална документация за Learning Management System (LMS) - система за управление на обучение. Този документ ще бъде актуализиран с напредъка на разработката.

- [Настройка на средата](#настройка-на-средата)
- [Архитектурни решения](#архитектурни-решения)

## Първи стъпки

### Изисквания
- Node.js (версия 16.x или по-нова)
- npm (версия 8.x или по-нова)
- Git
- Docker

### Клониране на хранилището
```bash
git clone [repository-url]
cd lms
```

## Технически стек (планиран)

### Backend (предложен)
- Node.js с TypeScript
- NestJS
- PostgreSQL за база данни
- JWT за автентикация

### Frontend (предложен)
- Codux (React с TypeScript)
- Material-UI или подобна UI библиотека
- React Query или подобно решение за управление на състоянието

## Настройка на средата

### Конфигурационен файл
Създайте `.env` файл в основната директория с необходимите променливи за средата. Примерна структура:

```env
# Основна конфигурация
NODE_ENV=development
PORT=3000

# Конфигурация на базата данни (ще бъде добавена по-късно)
# DB_HOST=localhost
# DB_PORT=5432
# DB_USERNAME=
# DB_PASSWORD=
# DB_NAME=
```

## Архитектурни решения

### Микро услугов дизайн
В системата ще бъде разработена като с микро-услугова архитектура:
1. Сервиз за потребителска автентикация
2. Потребителски сървис
3. Сервиз за курсов
4. Сервиз за задачи и предавания
5. Сервиз за оценяване
6. Сервиз за реклами

## Принос
Приноса ще идва от реклами които ще се появяват в сайта. Но това ще стане едва, след като софтуера е завърнеш, работи и е поставено съдържание с налични учебни материали.
