2. Single-open Accordion (Only One Section Open)

Identify input values, edge cases and expected results for each scenario.
Run tests manually or via automation scripts against the Sandbox UI.
Document found issues, screenshots and reproduction steps.

3. Accordion with Nested Content

  • Verify all accordion sections are collapsed by default (where specified).
  • Verify clicking a header toggles only its own content.
  • Verify keyboard navigation (Tab/Enter/Space) works where applicable.
  • Verify content is visible/hidden correctly after multiple toggles.
