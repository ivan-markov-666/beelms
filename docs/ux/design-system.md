# beelms – UX Design System (MVP)

_Роля: UX/FE. Фаза: BMAD Solutioning._

## Цел

Минимални правила за UI консистентност за MVP.

## Принципи

- Преизползваеми UI компоненти (buttons, inputs, alerts).
- Консистентни състояния: loading/empty/error.
- Достъпност: семантични HTML елементи + фокус стилове.
